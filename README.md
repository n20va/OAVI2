Размеры окон: 3x3, 25x25
Исходные данные
Использована выборка "Жесть" с сайта slavcorpora.ru по указанию преподавателя.
Количество изображений: 3
Размеры окон метода: 3x3, 25x25
Параметры метода: k1=0.15, k2=0.2, gamma=2.0, вторичное окно = 3 * D
Исходные изображения сохранены локально в PNG, полутоновые и бинарные — в BMP.
Теоретические сведения
Согласно таблице вариантов, для варианта 11 требуется реализовать адаптивную бинаризацию Феня и Тана с окнами 3×3 и 25×25.

Обесцвечивание выполняется по взвешенной формуле яркости:

I(x, y) = 0.299 * R(x, y) + 0.587 * G(x, y) + 0.114 * B(x, y)
Для метода Феня и Тана вычисляются локальные характеристики изображения в основном и вторичном окнах:

m(x, y)  — локальное среднее в основном окне
s(x, y)  — локальное среднеквадратическое отклонение в основном окне
M(x, y)  — локальный минимум яркости в основном окне
R(x, y)  — локальный динамический диапазон по вторичному окну

alpha2(x, y) = k1 * (s(x, y) / R(x, y))^gamma
alpha3(x, y) = k2 * (s(x, y) / R(x, y))^gamma
T(x, y)      = (1 - alpha2(x, y)) * m(x, y) + alpha3(x, y) * M(x, y)
B(x, y)      = 255, если I(x, y) > T(x, y), иначе 0
